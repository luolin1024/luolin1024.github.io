激活idea#

通过finder找到应用程序后IntelliJ IDEA通过显示包内容进入/Contents/bin文件夹下， 修改idea.vmoptions

尾部添加

--add-opens=java.base/jdk.internal.org.objectweb.asm=ALL-UNNAMED
--add-opens=java.base/jdk.internal.org.objectweb.asm.tree=ALL-UNNAMED

-javaagent:/Users/luolin/locaDevTool/ja-netfilter/ja-netfilter.jar=jetbrains

注意：/Users/luolin/locaDevTool/ja-netfilter是我电脑中的位置，此处需要改为自己的电脑位置 打开idea，编辑VM Options，添加上面的参数

完成后点击注册，选择如下配置：

配置激活服务器：

https://jetbra.in/

参考： macOS如何给idea安装ja-netfilter插件-CSDN博客 ja-netfilter 发行版 - Gitee.com

JRebel 激活#

  1. :生成 GUID 的网址 https://www.guidgen.com/
  2. :用这个网址 + 生成的 GUID 激活 https://jrebel.qekang.com/

例如: https://jrebel.qekang.com/f11c305b-a88b-454b-8bb5-0c280f3b6257

  1. :设置离线模式 来防止失效

File -> Settings -> JRebel -> [Work offline]按钮
