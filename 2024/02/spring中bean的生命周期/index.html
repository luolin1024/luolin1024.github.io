еүҚиЁҖ#

иҝҷе…¶е®һжҳҜдёҖйҒ“йқўиҜ•йўҳпјҢжҳҜжҲ‘еңЁйқўиҜ•зҷҫеәҰзҡ„ж—¶еҖҷиў«й—®еҲ°зҡ„пјҢеҪ“ж—¶жІЎжңүзӯ”еҮәжқҘ(еӣ дёәиҮӘе·ұзңҹзҡ„еҫҲиҸң)пјҢеҗҺжқҘеңЁзҪ‘дёҠеҜ»жүҫзӯ”жЎҲпјҢзңӢеҲ°д№ҹжҳҜдёҖеӨҙйӣҫж°ҙпјҢзӣҙеҲ°зңӢеҲ°дәҶгҖҠSpring in actionгҖӢиҝҷжң¬д№ҰпјҢд№ҰдёҠжңүеҜ№BeanеЈ°жҳҺе‘Ёжңҹзҡ„еӨ§иҮҙи§ЈйҮҠпјҢдҪҶжҳҜжІЎжңүд»Јз ҒеҲҶжһҗпјҢжүҖд»Ҙе°ұиҮӘе·ұдёҠзҪ‘еҜ»жүҫиө„ж–ҷпјҢдёҖе®ҡиҰҒжҠҠиҝҷдёӘBeanз”ҹе‘Ҫе‘Ёжңҹеј„жҳҺзҷҪпјҒ зҪ‘дёҠеӨ§йғЁеҲҶйғҪжҳҜйӘҢиҜҒзҡ„Bean еңЁйқўиҜ•й—®зҡ„з”ҹе‘Ҫе‘ЁжңҹпјҢе…¶е®һжҹҘйҳ…JDKиҝҳжңүдёҖдёӘе®Ңж•ҙзҡ„Beanз”ҹе‘Ҫе‘ЁжңҹпјҢиҝҷеҗҢж—¶д№ҹйӘҢиҜҒдәҶд№ҰжҳҜе…·жңүзүҮйқўжҖ§зҡ„пјҢжңҖfresh зҡ„иө„ж–ҷиҝҳжҳҜжҹҘйҳ…еҺҹе§ӢJDK!!!

дёҖгҖҒBean зҡ„е®Ңж•ҙз”ҹе‘Ҫе‘Ёжңҹ#

еңЁдј з»ҹзҡ„Javaеә”з”ЁдёӯпјҢbeanзҡ„з”ҹе‘Ҫе‘ЁжңҹеҫҲз®ҖеҚ•пјҢдҪҝз”ЁJavaе…ій”®еӯ— new иҝӣиЎҢBean зҡ„е®һдҫӢеҢ–пјҢз„¶еҗҺиҜҘBean е°ұиғҪеӨҹдҪҝз”ЁдәҶгҖӮдёҖж—ҰbeanдёҚеҶҚиў«дҪҝз”ЁпјҢеҲҷз”ұJavaиҮӘеҠЁиҝӣиЎҢеһғеңҫеӣһж”¶гҖӮ зӣёжҜ”д№ӢдёӢпјҢSpringз®ЎзҗҶBeanзҡ„з”ҹе‘Ҫе‘Ёжңҹе°ұеӨҚжқӮеӨҡдәҶпјҢжӯЈзЎ®зҗҶи§ЈBean зҡ„з”ҹе‘Ҫе‘ЁжңҹйқһеёёйҮҚиҰҒпјҢеӣ дёәSpringеҜ№Beanзҡ„з®ЎзҗҶеҸҜжү©еұ•жҖ§йқһеёёејәпјҢдёӢйқўеұ•зӨәдәҶдёҖдёӘBeanзҡ„жһ„йҖ иҝҮзЁӢ

Bean зҡ„з”ҹе‘Ҫе‘Ёжңҹ еҰӮдёҠеӣҫжүҖзӨәпјҢBean зҡ„з”ҹе‘Ҫе‘ЁжңҹиҝҳжҳҜжҜ”иҫғеӨҚжқӮзҡ„пјҢдёӢйқўжқҘеҜ№дёҠеӣҫжҜҸдёҖдёӘжӯҘйӘӨеҒҡж–Үеӯ—жҸҸиҝ°:

  1. SpringеҗҜеҠЁпјҢжҹҘжүҫе№¶еҠ иҪҪйңҖиҰҒиў«Springз®ЎзҗҶзҡ„beanпјҢиҝӣиЎҢBeanзҡ„е®һдҫӢеҢ–
  2. Beanе®һдҫӢеҢ–еҗҺеҜ№е°ҶBeanзҡ„еј•е…Ҙе’ҢеҖјжіЁе…ҘеҲ°Beanзҡ„еұһжҖ§дёӯ
  3. еҰӮжһңBeanе®һзҺ°дәҶBeanNameAwareжҺҘеҸЈзҡ„иҜқпјҢSpringе°ҶBeanзҡ„Idдј йҖ’з»ҷsetBeanName()ж–№жі•
  4. еҰӮжһңBeanе®һзҺ°дәҶBeanFactoryAwareжҺҘеҸЈзҡ„иҜқпјҢSpringе°Ҷи°ғз”ЁsetBeanFactory()ж–№жі•пјҢе°ҶBeanFactoryе®№еҷЁе®һдҫӢдј е…Ҙ
  5. еҰӮжһңBeanе®һзҺ°дәҶApplicationContextAwareжҺҘеҸЈзҡ„иҜқпјҢSpringе°Ҷи°ғз”ЁBeanзҡ„setApplicationContext()ж–№жі•пјҢе°ҶbeanжүҖеңЁеә”з”ЁдёҠдёӢж–Үеј•з”Ёдј е…ҘиҝӣжқҘгҖӮ
  6. еҰӮжһңBeanе®һзҺ°дәҶBeanPostProcessorжҺҘеҸЈпјҢSpringе°ұе°Ҷи°ғз”Ёд»–д»¬зҡ„postProcessBeforeInitialization()ж–№жі•гҖӮ
  7. еҰӮжһңBean е®һзҺ°дәҶInitializingBeanжҺҘеҸЈпјҢSpringе°Ҷи°ғз”Ёд»–д»¬зҡ„afterPropertiesSet()ж–№жі•гҖӮзұ»дјјзҡ„пјҢеҰӮжһңbeanдҪҝз”Ёinit-methodеЈ°жҳҺдәҶеҲқе§ӢеҢ–ж–№жі•пјҢиҜҘж–№жі•д№ҹдјҡиў«и°ғз”Ё
  8. еҰӮжһңBean е®һзҺ°дәҶBeanPostProcessorжҺҘеҸЈпјҢSpringе°ұе°Ҷи°ғз”Ёд»–д»¬зҡ„postProcessAfterInitialization()ж–№жі•гҖӮ
  9. жӯӨж—¶пјҢBeanе·Із»ҸеҮҶеӨҮе°ұз»ӘпјҢеҸҜд»Ҙиў«еә”з”ЁзЁӢеәҸдҪҝз”ЁдәҶгҖӮд»–д»¬е°ҶдёҖзӣҙй©»з•ҷеңЁеә”з”ЁдёҠдёӢж–ҮдёӯпјҢзӣҙеҲ°еә”з”ЁдёҠдёӢж–Үиў«й”ҖжҜҒгҖӮ
  10. еҰӮжһңbeanе®һзҺ°дәҶDisposableBeanжҺҘеҸЈпјҢSpringе°Ҷи°ғз”Ёе®ғзҡ„destory()жҺҘеҸЈж–№жі•пјҢеҗҢж ·пјҢеҰӮжһңbeanдҪҝз”ЁдәҶdestory-method еЈ°жҳҺй”ҖжҜҒж–№жі•пјҢиҜҘж–№жі•д№ҹдјҡиў«и°ғз”ЁгҖӮ

дёҠйқўжҳҜSpring дёӯBeanзҡ„ж ёеҝғжҺҘеҸЈе’Ңз”ҹе‘Ҫе‘ЁжңҹпјҢйқўиҜ•еӣһзӯ”дёҠиҝ°иҝҮзЁӢе·Із»Ҹи¶іеӨҹдәҶгҖӮдҪҶжҳҜзҝ»йҳ…JavaDocж–ҮжЎЈеҸ‘зҺ°йҷӨдәҶд»ҘдёҠжҺҘеҸЈеӨ–пјҢиҝҳжңүеҸҰеӨ–зҡ„еҲқе§ӢеҢ–иҝҮзЁӢж¶үеҸҠзҡ„жҺҘеҸЈпјҡ ж‘ҳиҮӘorg.springframework.beans.factory.BeanFactoryпјҢ е…ЁйғЁзӣёе…іжҺҘеҸЈеҰӮдёӢпјҢдёҠиҝ°е·Іжңүзҡ„е°ұдёҚз”ЁзқҖйҮҚж ҮжіЁпјҢжҠҠйўқеӨ–зҡ„зӣёе…іжҺҘеҸЈзқҖйҮҚж ҮжіЁдёӢ

Bean е®Ңж•ҙзҡ„з”ҹе‘Ҫе‘Ёжңҹ ж–Үеӯ—и§ЈйҮҠеҰӮдёӢпјҡ вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”еҲқе§ӢеҢ–вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”вҖ”

  • BeanNameAware.setBeanName() еңЁеҲӣе»әжӯӨbeanзҡ„beanе·ҘеҺӮдёӯи®ҫзҪ®beanзҡ„еҗҚз§°пјҢеңЁжҷ®йҖҡеұһжҖ§и®ҫзҪ®д№ӢеҗҺи°ғз”ЁпјҢеңЁInitializinngBean.afterPropertiesSet()ж–№жі•д№ӢеүҚи°ғз”Ё
  • BeanClassLoaderAware.setBeanClassLoader(): еңЁжҷ®йҖҡеұһжҖ§и®ҫзҪ®д№ӢеҗҺпјҢInitializingBean.afterPropertiesSet()д№ӢеүҚи°ғз”Ё
  • BeanFactoryAware.setBeanFactory() : еӣһи°ғжҸҗдҫӣдәҶиҮӘе·ұзҡ„beanе®һдҫӢе·ҘеҺӮпјҢеңЁжҷ®йҖҡеұһжҖ§и®ҫзҪ®д№ӢеҗҺпјҢеңЁInitializingBean.afterPropertiesSet()жҲ–иҖ…иҮӘе®ҡд№үеҲқе§ӢеҢ–ж–№жі•д№ӢеүҚи°ғз”Ё
  • EnvironmentAware.setEnvironment(): и®ҫзҪ®environmentеңЁз»„д»¶дҪҝз”Ёж—¶и°ғз”Ё
  • EmbeddedValueResolverAware.setEmbeddedValueResolver(): и®ҫзҪ®StringValueResolver з”ЁжқҘи§ЈеҶіеөҢе…ҘејҸзҡ„еҖјеҹҹй—®йўҳ
  • ResourceLoaderAware.setResourceLoader(): еңЁжҷ®йҖҡbeanеҜ№иұЎд№ӢеҗҺи°ғз”ЁпјҢеңЁafterPropertiesSet жҲ–иҖ…иҮӘе®ҡд№үзҡ„init-method д№ӢеүҚи°ғз”ЁпјҢеңЁ ApplicationContextAware д№ӢеүҚи°ғз”ЁгҖӮ
  • ApplicationEventPublisherAware.setApplicationEventPublisher(): еңЁжҷ®йҖҡbeanеұһжҖ§д№ӢеҗҺи°ғз”ЁпјҢеңЁеҲқе§ӢеҢ–и°ғз”ЁafterPropertiesSet жҲ–иҖ…иҮӘе®ҡд№үеҲқе§ӢеҢ–ж–№жі•д№ӢеүҚи°ғз”ЁгҖӮеңЁ ApplicationContextAware д№ӢеүҚи°ғз”ЁгҖӮ
  • MessageSourceAware.setMessageSource(): еңЁжҷ®йҖҡbeanеұһжҖ§д№ӢеҗҺи°ғз”ЁпјҢеңЁеҲқе§ӢеҢ–и°ғз”ЁafterPropertiesSet жҲ–иҖ…иҮӘе®ҡд№үеҲқе§ӢеҢ–ж–№жі•д№ӢеүҚи°ғз”ЁпјҢеңЁ ApplicationContextAware д№ӢеүҚи°ғз”ЁгҖӮ
  • ApplicationContextAware.setApplicationContext(): еңЁжҷ®йҖҡBeanеҜ№иұЎз”ҹжҲҗд№ӢеҗҺи°ғз”ЁпјҢеңЁInitializingBean.afterPropertiesSetд№ӢеүҚи°ғз”ЁжҲ–иҖ…з”ЁжҲ·иҮӘе®ҡд№үеҲқе§ӢеҢ–ж–№жі•д№ӢеүҚгҖӮеңЁResourceLoaderAware.setResourceLoaderпјҢApplicationEventPublisherAware.setApplicationEventPublisherпјҢMessageSourceAwareд№ӢеҗҺи°ғз”ЁгҖӮ
  • ServletContextAware.setServletContext(): иҝҗиЎҢж—¶и®ҫзҪ®ServletContextпјҢеңЁжҷ®йҖҡbeanеҲқе§ӢеҢ–еҗҺи°ғз”ЁпјҢеңЁInitializingBean.afterPropertiesSetд№ӢеүҚи°ғз”ЁпјҢеңЁ ApplicationContextAware д№ӢеҗҺи°ғз”ЁжіЁпјҡжҳҜеңЁWebApplicationContext иҝҗиЎҢж—¶
  • BeanPostProcessor.postProcessBeforeInitialization() : е°ҶжӯӨBeanPostProcessor еә”з”ЁдәҺз»ҷе®ҡзҡ„ж–°beanе®һдҫӢ еңЁд»»дҪ•beanеҲқе§ӢеҢ–еӣһи°ғж–№жі•(еғҸжҳҜInitializingBean.afterPropertiesSetжҲ–иҖ…иҮӘе®ҡд№үзҡ„еҲқе§ӢеҢ–ж–№жі•пјүд№ӢеүҚи°ғз”ЁгҖӮиҝҷдёӘbeanе°ҶиҰҒеҮҶеӨҮеЎ«е……еұһжҖ§зҡ„еҖјгҖӮиҝ”еӣһзҡ„beanзӨәдҫӢеҸҜиғҪиў«жҷ®йҖҡеҜ№иұЎеҢ…иЈ…пјҢй»ҳи®Өе®һзҺ°иҝ”еӣһжҳҜдёҖдёӘbeanгҖӮ
  • BeanPostProcessor.postProcessAfterInitialization() : е°ҶжӯӨBeanPostProcessor еә”з”ЁдәҺз»ҷе®ҡзҡ„ж–°beanе®һдҫӢ еңЁд»»дҪ•beanеҲқе§ӢеҢ–еӣһи°ғж–№жі•(еғҸжҳҜInitializingBean.afterPropertiesSetжҲ–иҖ…иҮӘе®ҡд№үзҡ„еҲқе§ӢеҢ–ж–№жі•)д№ӢеҗҺи°ғз”ЁгҖӮиҝҷдёӘbeanе°ҶиҰҒеҮҶеӨҮеЎ«е……еұһжҖ§зҡ„еҖјгҖӮиҝ”еӣһзҡ„beanзӨәдҫӢеҸҜиғҪиў«жҷ®йҖҡеҜ№иұЎеҢ…иЈ…
  • InitializingBean.afterPropertiesSet(): иў«BeanFactoryеңЁи®ҫзҪ®жүҖжңүbeanеұһжҖ§д№ӢеҗҺи°ғз”Ё(е№¶дё”ж»Ўи¶іBeanFactory е’Ң ApplicationContextAware)гҖӮ

  • е®һзҺ°дәҶиҮӘе®ҡд№үзҡ„destory()ж–№жі•

дәҢгҖҒBean зҡ„з”ҹе‘Ҫе‘ЁжңҹйӘҢиҜҒ#

дёәдәҶйӘҢиҜҒBeanз”ҹе‘Ҫе‘Ёжңҹзҡ„иҝҮзЁӢпјҢжңүдёӨз§ҚеҪўејҸпјҡдёҖз§ҚжҳҜдёәйқўиҜ•иҖҢеҮҶеӨҮзҡ„пјҢдёҖз§ҚжҳҜдёәдәҶи§Је…ЁиҝҮзЁӢиҖҢеҮҶеӨҮзҡ„пјҢдёӢйқўжқҘзңӢд»Јз Ғпјҡ Book.class

public class Book implements BeanNameAware,BeanFactoryAware,
        ApplicationContextAware,InitializingBean,DisposableBean {

    private String bookName;
    public Book(){
        System.out.println("Book Initializing ");
    }

    public void setBeanFactory(BeanFactory beanFactory) throws BeansException {
        System.out.println("Book.setBeanFactory invoke");
    }

    public void setBeanName(String name) {
        System.out.println("Book.setBeanName invoke");
    }

    public void destroy() throws Exception {
        System.out.println("Book.destory invoke");
    }

    public void afterPropertiesSet() throws Exception {
        System.out.println("Book.afterPropertiesSet invoke");
    }

    public void setApplicationContext(ApplicationContext applicationContext) throws BeansException {
        System.out.println("Book.setApplicationContext invoke");
    }

    public String getBookName() {
        return bookName;
    }

    public void setBookName(String bookName) {
        this.bookName = bookName;
        System.out.println("setBookName: Book name has set.");
    }

    public void myPostConstruct(){
        System.out.println("Book.myPostConstruct invoke");
    }

     // иҮӘе®ҡд№үеҲқе§ӢеҢ–ж–№жі•
    @PostConstruct
    public void springPostConstruct(){
        System.out.println("@PostConstruct");
    }

    public void myPreDestory(){
        System.out.println("Book.myPreDestory invoke");
        System.out.println("---------------destroy-----------------");
    }

    // иҮӘе®ҡд№үй”ҖжҜҒж–№жі•
    @PreDestroy
    public void springPreDestory(){
        System.out.println("@PreDestory");
    }

    @Override
    protected void finalize() throws Throwable {
        System.out.println("------inside finalize-----");
    }
}

иҮӘе®ҡд№үе®һзҺ°BeanPostProcessor зҡ„MyBeanPostProcessorпјҡ

public class MyBeanPostProcessor implements BeanPostProcessor {

    // е®№еҷЁеҠ иҪҪзҡ„ж—¶еҖҷдјҡеҠ иҪҪдёҖдәӣе…¶д»–зҡ„beanпјҢдјҡи°ғз”ЁеҲқе§ӢеҢ–еүҚе’ҢеҲқе§ӢеҢ–еҗҺж–№жі•
    // иҝҷж¬ЎеҸӘе…іжіЁbook(bean)зҡ„з”ҹе‘Ҫе‘Ёжңҹ
    public Object postProcessBeforeInitialization(Object bean, String beanName) throws BeansException {
        if(bean instanceof Book){
            System.out.println("MyBeanPostProcessor.postProcessBeforeInitialization");
        }
        return bean;
    }

    public Object postProcessAfterInitialization(Object bean, String beanName) throws BeansException {
        if(bean instanceof Book){
            System.out.println("MyBeanPostProcessor.postProcessAfterInitialization");
        }
        return bean;
    }
}
