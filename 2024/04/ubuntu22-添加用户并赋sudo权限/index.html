ubuntu22 添加用户并赋sudo权限

2024-04-09 · 1 min · 65 words · luolin 本文总阅读量 次 本文总访客量 人
  1. 添加用户
adduser username
  1. 添加sudo权限
usermod -G sudo username
  1. 添加root权限
sudo vim /etc/sudoers

# User privilege specification  
root ALL=(ALL) ALL  
username ALL=(ALL) ALL

保存退出，username 用户就拥有了root权限。
