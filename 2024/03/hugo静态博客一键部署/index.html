hugo静态博客一键部署

背景#

搭建了一个hugo+Papermod+github action自动部署的静态博客后，想着分享出来，可以帮助大家快速部署一个简洁好看的私人静态博客，支持他人评论，文章统计等功能，具备被google、baidu等搜索引擎索引等特点。

快速开始#

luolin1024/hugo-blog-starter: 用于通过vercel快速搭建一个hugo静态博客 (github.com) 点击use this template

vercel自动部署#

通过github账号登录vercel，选择刚刚的项目导入

github action方式自动部署#

.github/workflows/gh-pages.yml中的修改为自己的项目即可

  1. 自身可翻墙(github和vercel使用前提)

  2. 创建好github和vercel账号，这一步基本随便建

  3. 尽量自己有一个域名（可以在us.kg上申请一个），没有的话也没关系，就是国内无法访问而已，也可以考虑通过cloudflare加速

  4. 访问hugo-vercel-starter，可以快速部署自带doit主题静态博客，注意填写好hugo版本即可部署

  5. 最后，访问vercel给出的域名即可，我这里vercel给的域名hugo博客demo (demo-vercel-blog.vercel.app)

案例#

hugo博客demo (demo-vercel-blog.vercel.app)

自定义域名指向该静态博客#

前提：自己有一个域名 域名添加到cloudflare管理，并配置dns解析服务器，完成后即可添加对应的域名解析vercel给的xxx.xxx.app域名

上面cloudflare设置完成后在vercel里面添加自己的域名解析对应的app
