如何阅读一本书

2024-07-01 · 1 min · 472 words · luolin 本文总阅读量 次 本文总访客量 人
  1. 明确自己阅读的目的
    1. 扩充知识：期望扩充哪方面的知识呢
    2. 研究他人的方法论？
    3. 深挖某些知识点？
  2. 当前看的内容是否合适？
  3. 应用方法论
    1. SQ3R 阅读法
      1. Survey (预览): 快速浏览标题、目录、摘要等，了解文章或书籍的大意。
      2. Question (提问): 对内容提出问题，带着问题阅读能提高理解力。
      3. Read (阅读): 带着问题进行深入阅读。
      4. Recite (复述): 阅读后复述主要内容，加深记忆。
      5. Review (复习): 定期复习，巩固所学内容。
    2. PQRST 学习法
      1. Preview (预习): 预习学习材料，形成整体概念。
      2. Question (提问): 对学习材料提出问题。
      3. Read (阅读): 带着问题阅读，寻找答案。
      4. Self-recite (自我复述): 自我复述关键内容。
      5. Test (测试): 自我测试，检查理解和记忆效果。
    3. 费曼技巧
      • 选择一个你想学习的概念，然后试着用最简单的语言向别人解释。
      • 发现自己无法解释清楚的部分，回头再学习，直到能用简单的语言解释为止。
  4. 笔记整理：
    1. 边读边记: 在阅读过程中随时记录自己的思考和疑问。
    2. 整理笔记: 阅读结束后，将所有笔记整理成系统的知识框架，可以使用思维导图或者结构化笔记软件（如Notion）。
    3. 定期复习: 定期复习整理好的笔记，巩固记忆和理解。
    4. 作用：加强记忆、方便后期回顾
